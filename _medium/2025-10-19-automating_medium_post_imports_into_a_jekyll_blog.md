If you’re like me, writing on Medium but also maintain a Jekyll-powered personal site. Keeping both in sync manually is… tedious. Here’s a simple way to automate the process using Ruby and GitHub Actions.1. Pulling Medium RSS FeedMedium exposes an RSS feed for every user:https://medium.com/feed/@yourusernameIn Ruby, we can fetch and parse it using the built-in rss library:require &#39;rss&#39;require &#39;open-uri&#39;rss_url = &quot;https://medium.com/feed/@yourusername&quot;rss = RSS::Parser.parse(URI.open(rss_url).read, false)rss.items.each do |item| puts item.title # extract content, date, link, tags, etc.endHere’s how to clean up HTML and generate Markdown files ready for Jekyll:content_text = item.content_encoded.gsub(/&lt;[^&gt;]*&gt;/, &#39;&#39;).stripFile.write(&quot;_medium/#{item.title_slug}.md&quot;, content_text)Add metadata like date, tags, link, and even a preview image in the YAML front matter.2. Handling Tags and ImagesMedium’s RSS category fields hold tags. Images can be extracted from the &lt;img&gt; in the content:tags = item.categories.map(&amp;:content)img_match = content.match(/&lt;img.*?src=&quot;(.*?)&quot;/)image_url = img_match ? img_match[1] : &#39;https://via.placeholder.com/350x200&#39;3. Automating Updates with GitHub ActionsCreate a workflow in .github/workflows/medium_sync.yml:name: Sync Medium Postson: schedule: - cron: &quot;0 0 * * 0&quot; # every Sunday workflow_dispatch:jobs: update-medium: runs-on: ubuntu-latest steps: - uses: actions/checkout@v4 - uses: ruby/setup-ruby@v1 with: ruby-version: 3.2 - run: gem install rss - run: ruby _scripts/fetch_medium.rb - run: | git config user.name &quot;github-actions&quot; git config user.email &quot;github-actions@github.com&quot; git add _medium/ git diff - quiet || (git commit -m &quot;Update Medium posts&quot; &amp;&amp; git push)The _scripts/fetch_medium.rb script runs every week and commits new posts to the repo. GitHub Pages will automatically pick up the updates.4. Why It Works- RSS feed gives structured access to your posts, tags, and dates.- Ruby + RSS library keeps parsing simple.- GitHub Actions handles automation and publishing seamlessly.With this setup, my Jekyll blog finally can mirror my Medium content with minimal effort.
